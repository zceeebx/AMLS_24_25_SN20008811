# Datasets Folder

This folder is intended to store the datasets required for the classification tasks. By default, the folder is empty, and you need to manually add the datasets here.

## Instructions
1. Download the required datasets:
   - **BreastMNIST**: [Download from MedMNIST](https://medmnist.com/)
   - **BloodMNIST**: [Download from MedMNIST](https://medmnist.com/)
2. Extract the downloaded files and place them in this `Datasets/` folder.
